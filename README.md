# F1 Telemetry Analysis 🏎️

A Python project for analyzing and visualizing Formula 1 telemetry data. This project fetches real-time F1 data and provides insights into driver performance, speed comparisons, and racing analytics.

## Features

- 📊 Real-time F1 telemetry data analysis
- 🎨 Interactive visualizations and charts
- 🏁 Driver performance comparison
- ⚡ Speed analysis and performance metrics
- 💾 Data caching for efficient analysis

## Quick Start

### Prerequisites

- Python 3.8+
- Jupyter Notebook or JupyterLab

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mdtenseven/F1-Telemetry-Analysis.git
cd F1-Telemetry-Analysis
```

2. Create and activate a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

### Usage

Open the Jupyter notebook to explore F1 telemetry data:

```bash
jupyter notebook notebooks/telemetry_analysis.ipynb
```

Or use the quickstart script:

```bash
python quickstart.py
```
